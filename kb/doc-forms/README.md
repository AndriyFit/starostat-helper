# Формування документів — Ременівський старостинський округ

## Як формувати документи

1. Визначити тип документа
2. Зібрати необхідні дані (див. інструкцію до кожного типу)
3. Передати дані Claude або запустити генератор:
   ```
   node kb/docgen.js generate <id> '<json-дані>'
   ```
4. Готовий .docx буде у папці `output/`
